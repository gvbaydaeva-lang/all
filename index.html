Вы открыли файл с диска (file://). Так React и TypeScript не запускаются — нужен локальный адрес http://localhost:….

Самый простой вариант

npm install
npm run open:app

Скрипт соберёт проект и запустит просмотр через Vite — браузер откроется сам (адрес вида http://localhost:4178).

Режим разработки

npm run dev

Обычно откроется http://localhost:5173.

После настройки GitHub Pages (деплой из Actions) приложение будет по адресу gvbaydaeva-lang.github.io/all — его можно открыть как обычный сайт, без терминала.
