Этот HTML нельзя просто открыть двойным щелчком из папки. Браузер не собирает TypeScript/React сам — нужен локальный сервер Vite.

В терминале в папке проекта (один раз):

npm install
npm run dev

Команда dev поднимет сервер и сама откроет вкладку (http://localhost:5173) — интерфейс появится там.

После npm run build можно смотреть сборку: npm run preview — тоже через локальный адрес, не через file://.
